Gambler Run

title : Gambler's Run author: Preetam Balijepalli date: 25 May 2015

First Slide

A Shiny app that Simple app that lets you simulate a sequence of random events, for example coin flips, and plot the cummulative sum. This app allows you choose the odds of winning, the number of games to simulate, and the number of simulations to display simultaneously

The app has a table and a graph to be rendered

  • The app can be found : here
  • Source code can be found : here

How it looks like

Following can be determined

  • Average winning
  • Averate losses by moving the slider
    • odds of game
    • odds of winning
    • Number of runs

Following Libraries are Required

  • shiny
  • shinyIncubator
  • ggplot2

shiny::runGitHub('devdataproducts', 'balijepalli')
